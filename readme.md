# fridge
creates music videos from an image or gif and an audio track

## installation

```
git clone https://github.com/clippymusic/fridge
cd ./fridge
python3 -m pip install ./requirements.txt
```
---

### run from source
```
python3 ./fridge.py
```

## dependencies
- python3
- moviepy
